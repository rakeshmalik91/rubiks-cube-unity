**Cube Puzzle Simulator** is a 3D simulation of the classic Rubik's Cube puzzle, built with **Unity**. It features realistic rotation mechanics, customizable controls, and smooth animations.

## üéÆ Features
- **Realistic 3D Interaction**: Rotate the entire cube or individual layers with intuitive touch/mouse controls.
- **Customization**: Adjust rotation speed and touch sensitivity via settings.
- **Scramble**: Built-in randomization to shuffle the cube for a new challenge.
- **Move Counter**: Tracks your moves to help measure efficiency.
- **Save State**: Automatically remembers your settings (Speed, Sensitivity).

### Setup & Installation
1. Clone the repository.
2. Open the project folder `rubiks-cube (unity 6)` in **Unity Hub**.
3. Load the scene `Assets/Rubik/Rubik.unity`.
4. Press **Play** to test in the editor.
